# rest-basic
| ROUTE           | HTTP VERB | DESCRIPTION      |
|-----------------|-----------|------------------|
| /API/users      | GET       | get all data     |
| /API/users      | POST      | create a data    |
| /API/users/:id  | GET       | get a data       |
| /API/users/:id  | PUT       | update a data    |
| /API/users/:id  | DELETE    | delete a data    |
