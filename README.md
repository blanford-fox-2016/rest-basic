# rest-basic

| Routes          | HTTP           | Description  |
| --------------- |:-------------:| -----:|
| /api/users      | GET    | Get all the users
| /api/users/:id  | GET    | Get single users
| /api/users      | POST   | Create a user
| /api/users      | DELETE | Delete a user
| /api/users      | PUT    | Update a user with new info
