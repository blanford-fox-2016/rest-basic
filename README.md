# rest-basic
| ROUTE          | HTTP VERB | DESCRIPTION      |
|----------------|-----------|------------------|
| /api/datas     | GET       | get all data     |
| /api/datas     | POST      | create a data    |
| /api/datas/:id | GET       | get a data       |
| /api/datas/:id | PUT       | update a data    |
| /api/datas/:id | DELETE    | delete a data    |
